# Zyon AI System Specification

Zyon AI has two main subsystems.

## 1. Conversational AI Layer

- Chat interface
- Streaming responses
- Conversation memory
- Semantic retrieval

## 2. Monitoring Intelligence Layer

- Client monitoring
- Media monitoring
- Social monitoring
- Sentiment analysis
- Topic extraction
- PR recommendation engine

Monitoring is configuration-driven.

**Configuration file:** `config/clients.yaml`

**Example client:**

- **Sahi**
- **Competitors:** Zerodha, Upstox, Groww

Future modules will read this configuration.
